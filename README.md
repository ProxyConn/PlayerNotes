# PlayerNotes

##Installation:
- Turn off the server.
- Drag and drop plugin to BungeeCord folder.
- Turn on the server.
- Try out the /note command!

===
##Commands:
- `/note` - Main note command. Prints out help menu.
- `/note add` - Adds a note to a player.
- `/note list` - Lists notes on a player.
